Property Picker

This was my second project, when I first learned Ruby on Rails.

This is a Real Estate website where a customer can view the list of preperties available to buy and also the broker details for more info if needed

An Admin can upload properties, edit them or delete and also he can add a new broker, edit or delete.

Customers have the ability to view all the properties and also the respective broker details with a dynamic search Option, where they can search the properties based on the 'No of bedrooms' and 'Location' i.e. suburbs.

I worked on this system for 1 week.
